# mimir-apps-sdk

This repo is sdk to integrate third-party applications(Apps) with Mimir Wallet (https://app.mimir.global/).

Nearly compatible with all methods of [polkadot-js/extension](https://github.com/polkadot-js/extension), you can easily integrate it into existing apps.
